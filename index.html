Currency formatting function

Given a numeric input (or string that can be converted using parseFloat()), this function returns a currency-formatted string. The currency symbol, major unit separator, decimal separator and printing behaviour of minor units can be controlled. The function signature is:

formatCurrency(amount,
               prefix = '£',
               displayMinorUnitsLimit = 10000,
               majorSep = ',',
               decimalSep = '.')

The displayMinorUnitsLimit parameter sets the value below which minor currency units (pence, cents) will be displayed. Input is rounded to the nearest major or minor unit depending on whether displayMinorUnitsLimit has been exceeded or not.
