Currency formatting function

Apply currency formatting to numbers. The function signature is:

formatCurrency(amount,
               prefix = '£',
               displayMinorUnitsLimit = 10000,
               majorSep = ',',
               decimalSep = '.')

The displayMinorUnitsLimit parameter sets the value below which minor currency units (pence, cents) will be displayed. The function rounds fractional numbers to two decimal places.
